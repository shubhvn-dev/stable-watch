# StableWatch
### Stablecoin Depeg Risk Monitor — FRE 6871 Final Project

Can macroeconomic stress indicators and stablecoin price behavior predict 
depeg events before they occur?

## Files
- StableWatch.R      — Main analysis pipeline
- StableWatch.Rmd    — R Markdown report source
- shiny/             — Interactive dashboard

## Data Sources
- CoinGecko API (coingecko.com/en/api)
- FRED API (fred.stlouisfed.org)
